폰광판이란?

폰광판은 축구, 농구, 탁구, 테니스, 파크골프 등 다양한 스포츠 경기에서 실시간으로 점수를 기록하고 관리할 수 있는 무료 웹 앱입니다. 별도의 앱 설치 없이 브라우저에서 바로 실행되며, 스마트폰·태블릿·PC 모든 기기에서 완벽하게 동작합니다.

동네 풋살 경기, 학교 체육 수업, 동호회 대회, 가족 게임 등 어떤 상황에서도 폰광판 하나로 점수 관리를 손쉽게 할 수 있습니다. 직관적인 인터페이스와 빠른 반응 속도로 경기 진행에 방해 없이 점수를 즉시 기록할 수 있습니다.

음성 인식 기능으로 손을 사용하지 않고도 "홈 득점", "어웨이 2점" 같은 음성 명령으로 점수를 입력할 수 있으며, 전술 구상을 위한 드래그 앤 드롭 작전판 기능도 함께 제공합니다.

주요 기능

⚡

실시간 점수 관리

버튼 하나로 즉시 점수를 업데이트하며, 잘못 입력한 경우 -1 버튼으로 빠르게 수정할 수 있습니다.

⏱️

경기 시간 타이머

전/후반 시간을 타이머로 관리하고, 시간 종료 시 알람 소리로 알려줍니다. 원하는 시간으로 자유롭게 설정 가능합니다.

🎤

음성 인식 입력

Chrome 브라우저에서 음성으로 점수를 입력할 수 있습니다. "홈 득점", "어웨이 2점" 등의 명령을 인식합니다.

📋

전술 작전판

드래그 앤 드롭으로 선수를 배치하는 작전판 기능을 제공합니다. 축구 그라운드와 농구 코트 배경을 지원합니다.

⛳

파크골프 스코어카드

4인 파크골프 모드에서 18홀 전체 스코어를 관리합니다. 홀별 파(Par) 설정 및 누적 타수 자동 계산 기능을 제공합니다.

📱

모든 기기 지원

스마트폰 세로/가로 모드, 태블릿, PC 화면 크기에 완벽하게 맞춰지는 반응형 디자인을 적용했습니다.

사용 방법

  1. 1

    종목 선택

    화면 상단의 드롭다운 메뉴에서 원하는 스포츠 종목을 선택합니다. 선택한 종목에 따라 점수판 형태가 자동으로 변경됩니다.

  2. 2

    팀 이름 설정

    좌측의 'HOME', 우측의 'AWAY' 텍스트를 클릭하여 팀 이름을 자유롭게 수정할 수 있습니다. 최대 10자까지 입력 가능합니다.

  3. 3

    점수 입력

    각 팀의 +1 버튼을 눌러 점수를 올리고, -1 버튼으로 잘못 입력된 점수를 취소합니다. 농구 모드에서는 +2, +3 버튼도 제공됩니다.

  4. 4

    타이머 사용

    중앙의 타이머 숫자를 클릭하면 시간을 직접 입력할 수 있습니다. '시작' 버튼을 눌러 카운트다운을 시작하고, 시간 종료 시 자동으로 알람이 울립니다.

  5. 5

    작전판 활용

    하단의 작전판에서 선수 수를 입력하고 '적용' 버튼을 누르면 선수들이 배치됩니다. 원하는 위치로 드래그하여 경기 전략을 구상하세요.

지원 종목

⚽

축구 / 풋살

전/후반 타이머와 전술 작전판 기능을 제공합니다. 11인제 축구부터 5인제 풋살까지 모든 형태에 사용 가능합니다.

🏀

농구

1점, 2점, 3점 버튼과 쿼터 타이머를 제공합니다. 농구 코트 배경의 작전판으로 팀 전략을 수립할 수 있습니다.

🏓

탁구

세트 점수 관리 기능을 제공합니다. 단식·복식 경기 모두 사용 가능하며 빠른 점수 입력이 가능합니다.

🎾

테니스

0·15·30·40·AD 포인트 자동 전환 및 세트 점수 관리를 지원합니다. 테니스 점수 체계를 정확히 반영합니다.

⛳

파크골프

4인 전용 모드로 18홀 스코어카드를 제공합니다. 홀별 파(Par) 설정과 각 플레이어의 총 타수를 자동으로 계산합니다.

자주 묻는 질문

앱을 설치해야 하나요?

아니요, 별도의 앱 설치 없이 브라우저에서 바로 사용할 수 있습니다. 즐겨찾기에 추가하시거나 홈 화면에 바로가기를 만들면 더 편리하게 접속할 수 있습니다.

사용 요금이 있나요?

완전 무료로 제공됩니다. 회원가입도 필요 없습니다. 서비스 운영을 위해 광고가 표시될 수 있으나 서비스 이용에는 영향이 없습니다.

인터넷이 없어도 사용할 수 있나요?

처음 페이지를 로딩한 후에는 오프라인 환경에서도 기본 기능을 사용할 수 있습니다. 단, 음성 인식 기능은 인터넷 연결이 필요합니다.

음성 인식은 어떤 명령어를 사용하나요?

"홈 득점", "홈 1점", "에이 득점", "어웨이 2점", "블루 3점", "레드 감점", "취소" 등의 명령어를 지원합니다. Chrome 브라우저에서 가장 잘 작동합니다.

점수를 저장할 수 있나요?

현재 버전에서는 페이지를 새로 고침하면 점수가 초기화됩니다. 경기 중에는 화면이 꺼지지 않도록 기기 설정을 확인하시거나, 중요한 점수는 화면을 캡처해두시길 권장합니다.

파크골프 모드는 몇 명까지 지원하나요?

파크골프 모드는 최대 4인(P1~P4)까지 지원하며, 18홀 전체 스코어를 기록할 수 있습니다. 각 플레이어의 이름도 자유롭게 변경 가능합니다.
